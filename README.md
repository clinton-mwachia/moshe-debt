# Moshe Debt

**Moshe Debt** is a simple and powerful desktop application designed to help individuals track money they owe to other people. The app allows users to record debts, track payments, and automatically calculate remaining balances so that you always know exactly how much you still owe.

It is built for simplicity and clarity, making it ideal for personal use, freelancers, or small informal lending situations where accurate debt tracking is important.

---

## Installation Instructions

1. Download **moshe-debt.exe** from the releases.

2. Install the App and follow installation instructions.

3. start the app from start menu or desktop icon.

## Features

### 1. Add People (Creditors)

Users can add individuals or entities they owe money to.

Information stored includes:

* Name
* Amount

This allows you to keep a clear list of everyone you owe money.

---

### 2. Record Payments

Whenever you make a payment, you can record it against the debt.

Each payment includes:

* Debt reference
* Amount paid
* Balance

The system automatically updates the remaining balance.

---

### 3. Automatic Balance Tracking

Moshe Debt automatically calculates:

* Total amount owed
* Total amount paid
* Remaining balance

This prevents manual calculations and ensures accuracy.


## Example Workflow

1. Add a debt

```
Person: Jane
Amount: 10000
```

2. Record payments over time.

```
Payment 1: 3,000 KES
Payment 2: 2,000 KES
```

4. Remaining balance will automatically be calculated:

```
Total Debt: 10,000
Total Paid: 5,000
Balance: 5,000
```
